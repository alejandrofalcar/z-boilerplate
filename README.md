## Z Boilerplate

This is a boilerplate for [Next.js](https://nextjs.org/).

Contains:
- Prisma
- tRPC
- Next.js 15
- React 19
- Tailwind CSS
- TypeScript
- Biome
- Shadcn UI
- React Query
- Zod
- Better Auth
- Postgres

## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
```


Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
